Atributo rel (relationship)

O atributo rel em HTML significa "relationship" — ou seja, ele define o tipo de relação entre o documento atual e o link que está sendo referenciado.

noopener = Evita que a nova aba tenha acesso à aba original (mais seguro)
noreferrer = Evita que o site de destino veja de onde veio o clique
nofollow = Diz aos mecanismos de busca para não seguir o link
external = (esse sim é válido!) Indica que o link leva para fora do site

Atributo target

O atributo target é usado em links (a) e formulários (form) para definir onde o conteúdo do link ou da ação será exibido.

_self= (padrão) Abre o link na mesma aba/janela atual
_blank= Abre o link em uma nova aba ou janela
_parent= Abre o link na janela pai (em caso de frames)
_top= Abre no frame mais externo (remove qualquer iframe)
nomePersonalizado= Abre o link em uma janela com esse nome (pode ser usada como alvo em JavaScript)
